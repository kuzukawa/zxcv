Icon Design — Option B Refined

Speech bubble → terminal block, terminal green palette. Three proportion variants.

V1 — Floating marks

No outer container. Just the two shapes + arrow on transparent/dark background. Clean and airy. Best for dark mode READMEs.

V2 — App icon shape

Rounded square container (iOS/macOS app icon style). More structured. Works equally well in light and dark READMEs.

V3 — Wide banner

Horizontal pill container spanning README width. Shows an actual example command — users instantly understand the tool. Most informative of the three.

My pick: V2 (App icon shape) — square format is universal, renders well at any README width, and the contained shape reads as a proper logo mark rather than a diagram.
